# 📧 NLP Spam Detection (Mail + SMS Classification)

##  Overview

This project focuses on detecting **spam vs legitimate messages** using Natural Language Processing (NLP) and Machine Learning techniques.

It classifies both **emails and SMS messages** into:

* ✅ Spam
* ❌ Not Spam (Ham)

---

## 📂 Dataset

The dataset used for this project is available on Kaggle:

👉 https://www.kaggle.com/datasets/ogprakhar/spam-email-dataset

---

## 🧠 Features

* Text preprocessing (cleaning, tokenization)
* Stopword removal
* Vectorization using TF-IDF / CountVectorizer
* Machine Learning model training
* Evaluation using accuracy and metrics
